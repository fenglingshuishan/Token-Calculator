# Changelog

## 3.0.0-archive — 2026-07-15

- Marked the project as archived and maintenance-only.
- Reframed the supported use case around self-managed LLM API pipelines.
- Added honest precision labels, structural guards, compression-cost accounting,
  and break-even reporting.
- Added installable CLI entry points and packaged frontend assets.
- Added wheel/sdist, Docker, Docker Compose, and PyInstaller release paths.
- Added bilingual English and Simplified Chinese documentation.
- Documented that managed agents such as Codex do not expose enough of their
  assembled context for this project to optimize their true prompt cost.

## 2.0.0 — historical prototype

- Initial multi-tokenizer workbench, rule compressor, LLM compressor, and cost
  simulator prototype.
